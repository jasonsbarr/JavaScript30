JavaScript30: 30 Days of Simple Projects in JavaScript

JavaScript30 is a free course from designer/developer/entrepreneur Wes Bos to help aspiring JavaScript developers form a habit of daily practice and improve their skills with fun projects.

Since I have a difficult time with the whole "forming a habit" thing (because ADHD is fun!), I'm using this guided series as a way to get back into the habit of daily work as I undertake my self-designed Open Source Personal Computer Science Learning Program.
